# **BomberMan**



Cette application est une reproduction modifiée du jeu vidéo Bomberman.



Le jeu se déroule dans un labyrinthe parfait, de 2 à 5 joueurs sont à l'intérieur, l'objectif est d'être le dernier le en vie.

Pour éliminer les adversaires on utilise des bombes qui explosent en croix dans un rayon variable.

Il y a également des bonus et des murs destructibles



### Bonus :

F rouge : augmente la portée des bombes de celui qui le ramasse

B noir : augmente le nombre max de bombes de celui qui le ramasse

S jaune : augmente la vitesse de déplacement de celui qui le ramasse



### ATH:

Il y a en bas de la fenêtre du client une barre présentant la vie restante, le nombre de bombes restantes, et les modification de stats



### Commandes :

Z / ↑ : Aller en haut

Q / ← : Aller à gauche

S / ↓ : Aller en bas

D / → : Aller à droite

Espace : Placer une bombe
